# Client

Get fee rates

## Parameters

-   `protocol` **Protocol** Protocol to interact with. If there's no protocol provided, fee rates are retrieved from chain data providers

Returns **FeeRates** The fee rates (average, fast, fastest) in `Satoshis/byte`
